# Remote Script Launcher
This is a small app that's designed to be able to launch scripts remotely.

## Installation
Setup and build from the directory of the source code (`remote-script-launcher\`).

```batch
.\scripts\setup.bat && .\scripts\build.bat
```

An executable in the "`dist`" directory in the source code.

## Usage
Launch the executable, pick a target (`127.0.0.1` is selected by default), choose a script and what to do with it.
